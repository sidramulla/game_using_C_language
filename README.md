# C Adventure Game 🎮

A **beginner-friendly** command-line game built in C one of my first projects....

## How to Play
- You start with **3 lives** ❤️❤️❤️
- **5 levels** to complete
- Each level gives you **2 choices**
- Pick the wrong choice → lose 1 life
- Lose all lives → Game Over 💀
- Complete all 5 levels → You win! 🏆

## Concepts Used
- 🔢 Arrays
- 📦 Structures
- ✅ Conditional operators (if/else)
- ⚖️ Relational operators (==, !=, <, >)
- 🔄 Loops (while/for)
- ❤️ Life tracking system
